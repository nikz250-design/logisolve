# Configuración recomendada de Obsidian

## 1. Plugins recomendados

### Obsidian Git (sincronización automática)
1. Instalar desde **Settings → Community plugins → Obsidian Git**
2. Configurar:
   - `Vault backup interval (minutes)`: 5
   - `Pull updates on startup`: ✓
   - `Disable push`: ✗ (si quieres contribuir desde Obsidian)
3. Apuntar el vault a la carpeta `/logisolve/docs/`

### Dataview (tablas dinámicas)
Permite queries tipo SQL sobre los metadatos frontmatter. Ejemplo:
````
```dataview
TABLE actualizado, tags
FROM "negocio"
SORT actualizado DESC
```
````

---

## 2. Estructura del vault

```
logisolve/docs/
├── 00-MOC.md           ← Empieza aquí
├── tecnico/
├── negocio/
└── referencia/
```

Configura la carpeta raíz del vault en `/logisolve/docs/` o en `/logisolve/`  
(si la usas como subcarpeta, los wikilinks funcionan igual).

---

## 3. Cómo mantener la base actualizada

Cada vez que se hace un cambio importante en la app, Claude Code actualiza los archivos `.md` correspondientes y hace commit al repo. El plugin Obsidian Git lo jala automáticamente.

Para actualizar manualmente: `Cmd/Ctrl + P → Obsidian Git: Pull`
